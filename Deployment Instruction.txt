1. Download XAMPP 
2. Ensure database connection for MySQL and XAMPP is working
3. If MySQL in XAMPP control panel forcefully shutdown, navigate to task manager and find mysqld and close it. If further issues persist, it is advisable to go to config to access the php.ini file and change the port numbers
4. Create itproghs as the name of the database and import the database schema provided "itproghs.sql"
5. Access through browser
